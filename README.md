## Development

1. Install deps: `npm install`
2. Run dev server: `npm run dev`
3. Run tests: `npm test`
4. Validate track: `node tools/validateTrack.mjs`
5. Coverage: `npm run test:coverage`
6. Backend stack (containers): `npm run backend:up`
